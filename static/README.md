# Yunus Gafar Olabisi - Portfolio Website

Welcome to my personal portfolio website! This is a showcase of my work as a **UI/UX Designer & Frontend Developer**.

## 📋 Overview

This portfolio website is built with vanilla HTML, CSS, and JavaScript. It features a clean, modern design with responsive layouts that work across all device sizes.

## 🗂️ Project Structure

```
├── index.html          # Home/Landing page
├── index.css           # Home page styles
├── about.html          # About me page
├── about.css           # About page styles
├── project.html        # Projects showcase page
├── project.css         # Projects page styles
├── project.js          # Project interaction & modal functionality
├── contact.html        # Contact information page
├── contact.css         # Contact page styles
├── hero.jpg            # Hero section image
├── ptf image/          # Portfolio images folder
│   └── preview/        # Project preview images
└── README.md           # This file
```

## 📄 Pages

### Home (index.html)

- Hero section with introduction
- Navigation menu with hamburger menu for mobile
- Quick overview of professional identity

### About (about.html)

- Personal background and professional summary
- Skills and expertise highlights

### Projects (project.html)

- Portfolio of completed projects
- Interactive image modal for viewing project details
- Project preview images with click-to-expand functionality

### Contact (contact.html)

- Contact information and ways to get in touch

## 🎨 Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Hamburger Menu** - Mobile-friendly navigation
- **Image Modal** - Click on project images to view larger previews
- **Modern Styling** - Clean, professional aesthetic
- **Smooth Interactions** - JavaScript-powered interactive elements

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. Navigate through the menu to explore different sections
3. Click on project images to view larger previews
4. Use the contact page to get in touch

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Responsive styling and layouts
- **JavaScript (Vanilla)** - Interactive features and modal functionality

## 📱 Responsive Breakpoints

The website is optimized for:

- Mobile devices (320px+)
- Tablets (768px+)
- Desktop (1024px+)
